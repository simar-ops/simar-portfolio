# Simarpreet Singh | DevOps Hub & Portfolio

This repository contains the source code for my central portfolio hub hosted at **[simarpreet.in](https://simarpreet.in)**. It serves as the primary gateway to my professional identity, technical projects, and infrastructure services.

## 🚀 Infrastructure Overview

This project is part of a "Hub-and-Spoke" architecture designed to showcase full-stack deployment and DNS management skills.

- **Main Hub (This Repo):** [simarpreet.in](https://simarpreet.in) (Hosted on Vercel)
- **Service Spoke:** [monitor.simarpreet.in](https://monitor.simarpreet.in) (Infrastructure Health Monitor)
- **Backend Spoke:** [api.simarpreet.in](https://api.simarpreet.in) (Centralized API hosted on Render)

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Dark Theme)
- **Typography:** Lexend (Geometric Sans-Serif)
- **Deployment:** Vercel CI/CD
- **DNS Management:** BigRock (A Records, CNAMEs, and TXT Verification)

## 📂 Key Features

- **Project Showcase:** Interactive cards linking to live subdomains and documentation.
- **Responsive Design:** Optimized for mobile, desktop, and terminal-style viewing.
- **DevOps Focus:** Highlighting core competencies in Linux, Jenkins, Docker, and Networking.
